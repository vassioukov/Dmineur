Les services de votre carte Visa


Achetez en ligne en toute sérénité

Avec le service « Verified by Visa », les commerçants équipés peuvent vous demander de leur indiquer un code confidentiel à usage unique afin de renforcer la vérification de votre identité. Ce code vous est envoyé au moment de la transaction par SMS.

Paramétrez votre carte bancaire

Sur votre espace client, vous pouvez paramétrer votre cart 7j sur 7 et 24h sur 24. Faites opposition, demandez la modification de votre plafond, un retrait d'espèces exceptionnel ou encore la réédition de votre code confidentiel directement depuis votre Espace Client.
